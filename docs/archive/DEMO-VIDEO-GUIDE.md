# Demo Video Production Guide

Complete guide for creating Atlas Desktop demo videos.

---

## Video Specifications

### Short Demo (60 seconds)

- **Resolution:** 1920x1080 (1080p) or 3840x2160 (4K)
- **Frame Rate:** 60fps
- **Format:** MP4 (H.264 codec)
- **Audio:** 48kHz, stereo
- **Bitrate:** 15-20 Mbps (1080p) or 50-80 Mbps (4K)

### Feature Showcase (3 minutes)

- Same specifications as above
- Chapters/markers for each section

       TEXT: "Atlas Desktop"

## Recording Setup

### Hardware

- **Microphone:** USB condenser or lapel mic (Blue Yeti, Rode, etc.)
- **Display:** 1080p or 4K monitor for clean capture
- **Audio Interface:** Optional, for professional sound

### Software

- **Screen Recording:** OBS Studio (free), ScreenFlow (macOS), Camtasia
- **Audio Recording:** Audacity (for cleanup), Adobe Audition
- **Video Editing:** DaVinci Resolve (free), Premiere Pro, Final Cut Pro

### Environment

- Clean desktop background (dark preferred)
- No notifications or distractions
- Good lighting for any face-cam segments
- Quiet recording environment

---

## Recording Checklist

### Before Recording

- [ ] Clean desktop - hide icons, taskbar auto-hide
- [ ] Close unnecessary apps
- [ ] Disable notifications (Focus Assist on Windows, DND on macOS)
- [ ] Test microphone levels
- [ ] Configure Atlas with good API keys (fast responses)
- [ ] Practice the script 2-3 times
- [ ] Set screen resolution to target (1080p or 4K)
- [ ] Warm up voice (for Atlas TTS consistency)

### During Recording

- [ ] Record in sections (easier to edit)
- [ ] Leave 2-3 second pauses between sections
- [ ] Speak clearly and at moderate pace
- [ ] If you make a mistake, pause and redo that section
- [ ] Keep the orb visible and centered

### After Recording

- [ ] Review all footage
- [ ] Note timestamps of good takes
- [ ] Backup raw footage before editing

## Editing Guidelines

### Visual Style

- Clean cuts (avoid fancy transitions except for section changes)
- Subtle fade transitions between chapters
- Text overlays: white or cyan on dark backgrounds
- Atlas brand colors: Cyan (#00FFFF), Purple (#9966FF), Orange (#FF8844)

### Audio

- Background music: 10-15% volume (not distracting)
- Voice-over: Clear and prominent
- Atlas responses: Natural volume
- Sound effects: Subtle, if any

### Pacing

- Don't rush - let features breathe
- Quick cuts for montage sequences
- Slower pace for demonstrations
- Natural pauses for readability

### Captions

- Add closed captions for accessibility
- Use same font as branding
- Position at bottom, don't obscure UI

## Export Settings

### YouTube/General Upload

- Container: MP4
- Video Codec: H.264 (High Profile)
- Resolution: 1920x1080 or 3840x2160
- Frame Rate: 60fps
- Bitrate: 15-20 Mbps (1080p) or 50-80 Mbps (4K)
- Audio Codec: AAC
- Audio Bitrate: 320 kbps
- Audio Sample Rate: 48 kHz

### Twitter/X

- Max 2:20 duration
- 1280x720 recommended
- Max 512 MB file size

### LinkedIn

- Max 10 minutes
- 1920x1080
- Max 5 GB file size
